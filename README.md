# Usage
<!-- usage -->
```sh-session
$ npm install -g ekko-cli
$ ekko COMMAND
running command...
$ ekko (-v|--version|version)
ekko-cli/0.0.0 darwin-x64 node-v14.15.0
$ ekko --help [COMMAND]
USAGE
  $ ekko COMMAND
...
```
